GlobalMessageServiceGoogleCloudMessagingHelper

public protocol GlobalMessageServiceGoogleCloudMessagingHelper: class

Google Cloud Messaging helper protocol - Handles UIApplicationDidBecomeActiveNotification & UIApplicationDidEnterBackgroundNotification to connect/dissconnect GCMService - Handles receiving Apple Device token, and remote notification

  • didBecomeActive()

    Responser for UIApplicationDidBecomeActiveNotification. Connects GCMService to Google Cloud Messaging server
